## 📋 Sobre

Site institucional de uma barbearia com design elegante em tons escuros e dourados, totalmente responsivo e com navegação suave entre seções.

## 🚀 Funcionalidades

- **Header fixo** com navegação e ícones sociais
- **Hero** em tela cheia com imagem de fundo e call-to-action
- **Quem Somos** — seção institucional com layout em duas colunas
- **Serviços** — catálogo de preços com combos
- **Trabalhos** — galeria de fotos em grid com efeito hover
- **Localização** — mapa integrado do Google Maps
- **Menu hamburger** responsivo com animação
- **Botão do WhatsApp** flutuante com animação pulse
- **Footer** com contatos, redes sociais e horários

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (Flexbox, Grid, animações, variáveis CSS)
- JavaScript vanilla
- Font Awesome 6 (ícones)
- Google Maps Embed

## 📱 Responsivo

Layout adaptado para dispositivos móveis com menu hamburger funcional.

## 📂 Estrutura

```
├── index.html    # Página principal
├── style.css     # Estilos
├── script.js     # Menu mobile e interações
```
